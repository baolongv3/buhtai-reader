# Buhtai-reader

This app is used for reading buhtai

## Installation

Use the package manager [npm](https://www.youtube.com/watch?v=dQw4w9WgXcQ) to install dependencies.

```bash
npm install
```

## Usage

```
ng serve -o 
```
